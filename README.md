# 메리츠 암보장 분석기 (Cancer Treatment Analysis Tool)

메리츠 암보장 분석기는 보험 가입 제안서(PDF)를 분석하여 암 통합치료비 및 각종 보장 내역을 추출하고 시각화해주는 전문가용 도구입니다.

## 주요 기능
- **제안서 분석**: PDF 파일을 업로드하면 AI와 정규식을 통해 핵심 보장 내역을 자동으로 추출합니다.
- **실시간 대시보드**: 추출된 데이터를 바탕으로 고객에게 보여줄 수 있는 깔끔한 분석 화면을 렌더링합니다.
- **매니저 환영 시스템**: 업로드된 파일명이나 매니저 정보를 인식하여 맞춤형 환영 메시지와 레벨Badge를 표시합니다.
- **결과 저장 및 공유**: 분석된 결과는 이미지 또는 PDF로 즉시 내보낼 수 있습니다.
- **데이터 동기화**: Supabase를 연동하여 실시간으로 데이터를 관리하고 분석 카운트를 추적합니다.

## 기술 스택
- **Frontend**: HTML5, Vanilla JS, Tailwind CSS
- **Libraries**:
  - `pdf.js`: PDF 텍스트 추출
  - `html2canvas`, `jspdf`: 결과물 이미지/PDF 저장
  - `Supabase SDK`: 백엔드 DB 및 실시간 기능
- **Backend/Logic**: `js/analyzer.js`, `js/ui_renderer.js`

## 설치 및 실행
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html`을 브라우저에서 엽니다. (로컬 서버 환경 권장)
3. `manager.txt`를 통해 매니저 정보를 관리할 수 있습니다.
